CRUD de Leitura de Livros

Organização simples para cadastro e gerenciamento de livros lidos.

Estrutura do Banco de Dados

Use este esquema para criar a tabela principal de livros.

CREATE TABLE livros (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    titulo TEXT NOT NULL,
    autor TEXT NOT NULL,
    status TEXT NOT NULL,
    nota INTEGER
);
CREATE TABLE sqlite_sequence(name,seq);

Visão Geral do CRUD

Cadastrar Livro

Adicionar um novo livro com título, autor, status e nota.

Listar Livros

Exibir todos os livros cadastrados e seus detalhes.

Editar Livro

Atualizar informações de um livro existente.

Excluir Livro

Remover um livro do registro quando necessário.
